# Gaedening_Diary

# 環境構築
1. リポジトリをクローンする  
```git clone git@github.com:matsuri002/Gaedening_Diary.git```
2. npm installの方法  
```cd Gaedening_Diary```  
```cd my-app```  
```npm install```  
3. requirements.txtのインストール  
```cd Gaedening_Diary```   
```pip install -r requirements.txt```

# ローカルで動かす手順  
＊ FastAPIとReactはそれぞれ別のターミナルで立ち上げる
## Reactの起動
1. ディレクトリをGaedening_Diaryに移動する  
```cd Gaedening_Diary```
2. ディレクトリをmy-appに移動する  
```cd my-app```
3. FastAPIの起動  
```npm start```
